Dasar 
  • Klik ikon aplikasi dan secara default, halaman Laporan Statistik akan terbuka
  • Mulailah dengan menambahkan penghasilan dengan cara klik icon menu dan Tambahkan Penghasilan
  • Mulailah dengan menambahkan pengeluaran dengan cara klik icon menu dan Tambahkan Pengeluaran
  • Untuk melihat riwayat penghasilan klik icon menu dan pilih Riwayat Penghasilan
  • Untuk melihat Riwayat Pengeluaran klik icon menu dan pilih Riwayat Pengeluaran
  • Untuk melihat statistik laporan klik icon menu dan pilih Statistik Laporan
  • Untuk mengatur data yang ditampilkan klik icon menu dan pilih Pengaturan
Melihat, Mengedit, dan Menghapus catatan
Untuk menghapus catatan baik itu catatan pemasukan maupun catatan pengeluaran klik ikon menu dan pilih riwayat pemasukan/pengeluaran lalu klik salah satu catatan yang tersedia. Akan muncul tiga menu sebagai berikut:
  • Lihat catatan : Untuk melihat catatan
  • Edit catatan : Untuk mengedit data dalam catatan
  • Hapus catatan: Untuk menghapus catatan secara permanen dari database aplikasi

Membackup data
Untuk fitur back up saat ini belum tersedia, namun kami terus berusaha untuk menambahkan fitur ini di masa mendatang.

Catatan penting
  • Data yang di hapus akan terhapus secara permanen
  • Jika anda uinstal aplikasi maika data anda akan terhapus secara keseluruhan
  • Proses upgrade mungkin akan menyebabkan aplikasi mengalami gangguan (tidak bisa di buka), uinstal aplikasi dan download kembali dari playstore
