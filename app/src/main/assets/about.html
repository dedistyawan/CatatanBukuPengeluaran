Aplikasi ini dalam tugas Pemrograman Mobile

 Hak Cipta dan Konten Pihak Ketiga
kami menggunakan Library Terbuka (Open Source Library) untuk membangun beberapa fungsi dari aplikasi ini, daftar library tersebut:

  • Yalantis - Context Menu
  • Blackfizz - EazeGraph
  • Nightonke - BoomMenu
  • Material Date Time Picker
  • Material Edit Text
  • Fbutton - Material Button
  • Material Icon by Google
Library berada dalam lisensi sumber terbuka yang bebas untuk digunakan dan disebarluaskan dengan atau tanpa modifikasi namun tidak mencakup untuk menjual ulang

Terimakasih untuk pihak terkait yang telah menjadikan library tersebut gratis dan bebas untuk digunakan
